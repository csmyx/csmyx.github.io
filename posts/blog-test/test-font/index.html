字体显示测试#

中文部分#

你好！这是一段用于测试中文字体显示的文本。
这里包含常见汉字、标点符号（。，；：“”‘’《》？！），以及全角字符和半角字符的混合使用。
请检查字体是否清晰、字形是否统一，尤其是标点符号的显示是否与文字风格匹配。
例如：“Hugo 静态博客系统”中的中英混合词汇，是否能正确渲染且无乱码。

英文部分#

Hello! This is a test for English font display.
Including uppercase/lowercase letters (A-Z, a-z), numbers (0-9), symbols (@ # $ % & *), and common phrases.
Check if the font kerning, spacing, and punctuation (such as commas, periods, apostrophes) look natural.
Example: “GitHub Actions workflow” should show consistent style for both letters and symbols.

混合段落#

当中英文混合时（如：在 Hugo 主题中配置 font-family 后），需确保两种文字的字体搭配协调，无突兀的大小或风格差异。
例如：“Please visit 我的博客 (My Blog) 查看更多内容！” 中的中英文应分别使用预设的字体，且排版整齐。
